# Changelog

All notable changes to Deal Sorter and Analyzer will be documented in this file.

## [0.1.0] - 2026-03-25

### Added
- Initial project scaffold generated by Project Generator
- Next.js 15 application with App Router and TypeScript strict mode
- Supabase integration with PostgreSQL database and authentication
- Complete data model with 12 tables:
  - users table with role-based access (Analyst, Partner, Associate)
  - companies table for target and comparable companies
  - deals table for M&A transaction tracking
  - financial_data table for company financial metrics
  - financial_ratios table for calculated PE metrics
  - comparable_analyses table for comp analysis results
  - deal_scoring_criteria table for customizable scoring factors
  - deal_scores table for deal evaluation results
  - historical_transactions table for precedent analysis
  - financial_models table for DCF/LBO model storage
  - integration_configs table for external service settings
  - sync_logs table for integration audit trail
- Row Level Security (RLS) policies for all database tables
- Authentication system with role-based authorization
- 23 route stubs matching complete site map:
  - Public routes: /, /login, /signup, /forgot-password
  - Protected dashboard routes: /dashboard, /deals, /deals/:id, /import
  - Analysis routes: /analysis, /analysis/comps, /analysis/comps/:id, /analysis/precedents, /analysis/reports, /analysis/reports/:id
  - Scoring and export routes: /scoring, /export, /export/:id, /reports, /settings
  - API routes: /api/deals, /api/import/salesforce, /api/analysis/comps, /api/export/quickbooks
- Integration stubs for 7 external services:
  - Salesforce CRM for deal data import
  - QuickBooks for financial model export
  - Google Sheets for spreadsheet integration
  - Airtable for flexible data management
  - Zapier for workflow automation
  - Notion for documentation and reporting
  - BigQuery for data analytics and reporting
- UI component library with Tailwind CSS:
  - Base UI components (buttons, forms, inputs, charts)
  - Authentication components (login, signup forms)
  - Dashboard layout and navigation
  - Deal management interface components
  - Analysis and comparison display components
- TypeScript type definitions generated from database schema
- Environment variable configuration for all integrations
- Complete documentation suite:
  - README.md with setup and deployment instructions
  - CLAUDE.md with AI coding session guidelines
  - TECHNICAL_DEBT.md with known limitations
  - ROADMAP.md with feature development plan
- Vercel deployment configuration
- ESLint and Prettier configuration for code quality
- Tailwind CSS configuration with custom theme
- Next.js configuration optimized for dashboard application
